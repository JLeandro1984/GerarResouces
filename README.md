## 📌 Resource Key Generator

Aplicação web simples e eficiente desenvolvida em **HTML, CSS e JavaScript puro**, com o objetivo de automatizar a geração de **chaves de resources (i18n)** a partir de frases.

A ferramenta permite que o usuário insira um texto e obtenha automaticamente três variações padronizadas amplamente utilizadas em desenvolvimento:

* **CONSTANT_CASE** → formato em caixa alta com underscores
* **PascalCase** → palavras concatenadas com iniciais maiúsculas
* **camelCase com prefixo** → padrão camelCase com prefixo configurado

---

## ⚙️ Funcionalidades

* Entrada de texto para geração de chaves
* Normalização automática (remoção de acentos e caracteres especiais)
* Conversão inteligente de termos (ex: `N°` → `Num`)
* Geração instantânea de múltiplos padrões de nomenclatura
* Botão de cópia rápida para cada resultado
* Interface moderna, responsiva e de fácil uso

---

## 🎯 Objetivo

Padronizar e acelerar a criação de identificadores de mensagens, reduzindo erros manuais e garantindo consistência em projetos que utilizam internacionalização ou controle de mensagens.

---

## 💻 Tecnologias
